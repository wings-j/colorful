# Colorful

取色工具。
链接：<a href='https://wings-j.github.io/colorful/' target='_blank'>Colorful</a>

## 主要功能

- 色轮取色
- RGB、HSL、HSV 数值调整，支持鼠标滚轮调整
- 16 进制显示
- 颜色暂存、比对
- CSS RGB、HSL、#语句输出

## 依赖技术

- HTML、CSS、Javascript
- Canvas、Service Worker
- React、Less
- Webpack

## 浏览器兼容性

- chrome:66.
- FireFox:60.
- Edge:16.
